# Mr. Auto Typer - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) - Professional, tech-focused
- **Secondary**: Electric blue (#00d4ff) - Modern, digital accent
- **Accent**: Warm amber (#ffb347) - Highlighting important elements
- **Background**: Off-white (#fafafa) - Clean, readable base
- **Text**: Rich black (#2d2d2d) - High contrast readability

### Typography
- **Display Font**: "JetBrains Mono" - Monospace for tech authenticity
- **Heading Font**: "Inter" - Modern, clean sans-serif
- **Body Font**: "Inter" - Consistent, highly readable
- **Code Font**: "JetBrains Mono" - For technical elements and demos

### Visual Language
- **Minimalist Tech Aesthetic**: Clean lines, purposeful whitespace
- **Subtle Gradients**: Gentle blue-to-amber gradients for depth
- **Geometric Patterns**: Grid-based layouts reflecting code structure
- **Soft Shadows**: Subtle elevation without harsh contrasts
- **Rounded Corners**: 8px radius for modern, approachable feel

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth animations for UI transitions and typing demos
- **p5.js**: Dynamic background particle system representing code flow
- **ECharts.js**: Data visualization for usage statistics and metrics
- **Splide.js**: Smooth carousel for feature showcases
- **Shader-park**: Subtle background shader effects

### Effect Implementation
- **Hero Background**: Animated particle field with code-like movement
- **Text Effects**: 
  - Typewriter animation for headlines
  - Color cycling emphasis on key features
  - Gradient text animation for call-to-actions
- **Interactive Elements**:
  - Smooth hover transitions with scale and glow
  - Real-time typing simulation with visual feedback
  - Progressive disclosure for feature details
- **Scroll Animations**: Subtle reveal effects as content enters viewport

### Header Effect
- **Floating Navigation**: Semi-transparent background with backdrop blur
- **Active State Indicators**: Smooth underline animations
- **Logo Animation**: Subtle pulse effect on the "Mr. Auto Typer" branding

### Styling Approach
- **Mobile-First**: Responsive design starting from 320px width
- **Component-Based**: Reusable UI elements with consistent styling
- **Performance-Focused**: Optimized animations and minimal asset loading
- **Accessibility-Compliant**: High contrast ratios and keyboard navigation support
